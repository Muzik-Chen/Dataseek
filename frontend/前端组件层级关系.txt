五、关键依赖链（业务闭环，含中文名称）
================================================================================

1. AI 聊天闭环:
   ChatWidget [AI聊天悬浮组件] → useSSE → chatStore → FoodCard [美食卡片] / HeritageCard [非遗卡片]
   (ChatWidget 内部 import FoodCard + HeritageCard，根据 SSE 消息 type 动态渲染)

2. 美食浏览闭环:
   App.vue → AppHeader [应用头部] (SearchBar [搜索栏]) → router-view →
     FoodList [美食推荐] (FoodCard [美食卡片]) → FoodDetail [美食详情] (FoodCard [美食卡片])

3. 非遗浏览闭环:
   App.vue → AppHeader [应用头部] (SearchBar [搜索栏]) → router-view →
     HeritageList [非遗民俗] (HeritageCard [非遗卡片]) → HeritageDetail [非遗详情]

4. 行程规划闭环:
   ChatWidget [AI聊天悬浮组件] (问AI) / TripCreate [创建行程] (手动表单) →
     MapContainer [地图容器] (全屏地图 + 路线绘制) + TripCard [行程方案卡片] →
     TripDetail [行程详情] (MapContainer [地图容器] + TripCard [行程方案卡片])

   7/10 UX 优化: 保存不跳转、离开确认、AI 生成完自动存草稿、侧边栏 hover 收起、
   地图路线持久化恢复、草稿状态标签（draft → 橙色"草稿"）

5. 后台管理闭环:
   App.vue → router-view → AdminLayout [管理后台布局] →
     AdminDashboardData [数据管理] (WeatherChart [天气图表] + CrowdChart [人流图表])
     ├── AdminFoods [美食管理] (ImageUploader [图片上传])
     └── AdminHeritages [非遗管理] (ImageUploader [图片上传])

6. 全局搜索闭环:
   AppHeader [应用头部] (SearchBar [搜索栏]) → /search → SearchResult [搜索结果]
   SearchOverlay [全局搜索遮罩] (Ctrl+K) → 各详情页
